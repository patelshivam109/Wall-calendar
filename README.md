# Wall Calendar Component

An interactive, responsive wall calendar built with Next.js and TypeScript.

## Features

- Month-based calendar view
- Date range selection (start → end)
- Reverse selection handling
- Notes tied to selected range
- LocalStorage persistence
- Responsive design (desktop + mobile)
- Hover preview for range selection

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- date-fns

## How to Run

```bash
npm install
npm run dev
